### LSP Autocompletion (nvim-cmp)

The configuration adds LSP-powered autocompletion using nvim-cmp, integrating with Neovimâ€™s built-in LSP client. It combines suggestions from:
- Active language servers (via `cmp-nvim-lsp`)
- Snippets (via `LuaSnip`)
- Current buffer words
- File system paths
- Optional icons/kinds (via `lspkind`)

Completion keybindings (Insert mode unless noted):
- `<C-Space>` â€” Manually trigger completion
- `<C-n>` / `<C-p>` â€” Select next/previous item
- `<C-d>` / `<C-f>` â€” Scroll completion docs up/down
- `<C-e>` â€” Abort/close the completion menu
- `<CR>` â€” Confirm selection (accept the highlighted item)
- `<Tab>` â€” Next item when the menu is visible; otherwise jump to next snippet placeholder
- `<S-Tab>` â€” Previous item when the menu is visible; otherwise jump to previous snippet placeholder

Tip: Neovimâ€™s built-in LSP omni-completion is also available via `<C-x><C-o>` in Insert mode.
